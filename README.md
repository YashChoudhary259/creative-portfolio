# Personal Portfolio Website 🚀

Welcome to my personal portfolio repository! This website serves as my digital sandbox and professional home on the web. It's designed to showcase who I am, the projects I've built, and the skills I use to bring ideas to life.

I built this portfolio to focus heavily on clean design, performance optimization, and a seamless mobile-first user experience.

## ✨ Core Sections

* **The Hero Section:** A punchy, conversational introduction welcoming visitors and summarizing my development focus.
* **Project Showcase:** A dynamic, card-based gallery highlighting my favorite projects (like QuizStorm and the Bus Management Ecosystem), complete with live links and source code.
* **Skill Inventory:** A visual, organized breakdown of my technical toolkit, from vanilla web technologies to modern frameworks.
* **Interactive Contact Portal:** A streamlined contact section making it effortless for recruiters, clients, or other developers to get in touch.

## 🛠️ The Tech Stack

I kept the stack modern, lightweight, and focused on clean engineering:

* **HTML5 / CSS3** – For a modular, component-driven layout and ultra-fast page load speeds.
* **Modern CSS** – Utilizing flexbox, grid layouts, and smooth animations to ensure a fluid, responsive design across all devices.
* **JavaScript (ES6+)** – Handling dynamic interactions, menu toggles, and state management.
